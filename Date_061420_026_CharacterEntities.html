There are certain characters which cannot be typed directly from keyboard (example, copyright symbol) and also there are certain characters which have special meaning in html files (example, less than symbol). Therefore, in order to use such characters, each known character in human language has been assigned a code which when written in html file appears as that character on the browser.

The list of characters and their code can be found here - https://dev.w3.org/html5/html-author/charref

Here is the 1/4 character: ¼

Here is the Integral character: ∫

Here is the Infinity character: ∞

Here is the telephone character: ☎

Here is the star character: ✶
