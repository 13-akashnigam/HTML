Tables in html are used for organizing data in the form of row and columns or tabular form. A table is represented using the table tag. table tag has two sub tags - thead i.e table header and tbody i.e table body. Each of these subtags have table rows repsented by tr tag and cells within row represented by th tag in case of thead tag and td tag in case of tbody tag. The difference between th tag and td tag for representing data in cells is that th bolds the text while td does not.
